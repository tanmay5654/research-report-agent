# Research & Report Agent

An agentic AI pipeline that researches any topic on the web and generates a structured professional report — fully automated.

## How It Works

```
User Topic → [Search] → [Analyze] → [Write Report] → Markdown File
```

1. **Search Node** — Searches the web using Tavily API (5 sources)
2. **Analyze Node** — LLM reads and extracts key insights from sources
3. **Report Node** — LLM writes a structured Markdown report

## Tech Stack

- **LangGraph** — agent orchestration and graph flow
- **Groq (Llama 3.3 70B)** — LLM for reasoning and writing
- **Tavily API** — web search for real-time information
- **Python** — core language

## Setup

1. Clone the repo
   ```bash
   git clone https://github.com/tanmay5654/research-report-agent.git
   cd research-report-agent
   ```

2. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```

3. Create `.env` file
   ```bash
   cp .env.example .env
   ```
   Fill in your API keys:
   - **Groq API key** → [console.groq.com](https://console.groq.com) (free)
   - **Tavily API key** → [app.tavily.com](https://app.tavily.com) (free tier)

4. Run the agent
   ```bash
   python agent.py
   ```
