# Research & Report Agent

A full-stack Agentic AI application that researches any topic on the web and generates a structured professional report — fully automated.

![Tech Stack](https://img.shields.io/badge/LangGraph-Agent-blue) ![Groq](https://img.shields.io/badge/Groq-Llama_3.3_70B-orange) ![FastAPI](https://img.shields.io/badge/FastAPI-Backend-green) ![React](https://img.shields.io/badge/React-Frontend-61dafb)

---

## How It Works

```
User Topic
    ↓
[Search Node]       → Tavily searches the web (5 sources)
    ↓
[Analyze Node]      → LLM extracts key insights from sources
    ↓
[Write Report Node] → LLM writes a structured Markdown report
    ↓
React UI displays the report + option to download
```

---

## Tech Stack

| Layer | Technology |
|---|---|
| Agent Orchestration | LangGraph |
| LLM | Groq — Llama 3.3 70B (free) |
| Web Search | Tavily API |
| Backend | FastAPI + Python |
| Frontend | React + Vite |

## Setup & Run

### 1. Clone the repo
```bash
git clone https://github.com/tanmay5654/research-report-agent.git
cd research-report-agent
```

### 2. Backend setup
```bash
cd backend
pip install -r requirements.txt
```

Create a `.env` file inside `backend/`:
```
GROQ_API_KEY=your_groq_api_key_here
TAVILY_API_KEY=your_tavily_api_key_here
```

Get free API keys:
- **Groq** → [console.groq.com](https://console.groq.com)
- **Tavily** → [app.tavily.com](https://app.tavily.com)

Start the backend:
```bash
uvicorn main:app --reload --port 8000
```

### 3. Frontend setup
```bash
cd ../frontend
npm install
npm run dev
```

Open **http://localhost:5173** in your browser.
